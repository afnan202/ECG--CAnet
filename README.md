ECG-CAnet

This study proposes an automated ECG arrhythmia classification framework using a hybrid CNN–attention architecture evaluated on the MIT-BIH Arrhythmia Database. ECG signals from 48 patient records were preprocessed through Wiener filtering, z-score normalization, and R-peak–based beat segmentation. Heartbeats were labeled into normal and abnormal classes following AAMI standards, resulting in 112,544 beats. A multi-domain feature extraction pipeline captured temporal, frequency, and wavelet characteristics using statistical measures, FFT-based spectral features, and DWT with db4 wavelets, forming a 28-dimensional feature vector per beat. The proposed deep learning model integrates residual 1D convolutional blocks with a multi-head self-attention mechanism to enhance discriminative feature learning and temporal focus. The network was trained using Adam optimization with learning rate scheduling, regularization, and early stopping. Performance was evaluated using accuracy, precision, recall, F1-score, AUC-ROC, and confusion matrices. Attention visualization and feature correlation analysis provided interpretability, enabling clinically meaningful insights alongside high classification accuracy.
